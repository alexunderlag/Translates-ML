Веб-приложение для автоматического перевода текста между русским и английским языками с использованием предобученных нейросетевых моделей из библиотеки **Transformers**.

## О проекте
Проект представляет собой простой онлайн-переводчик, работающий на базе моделей машинного перевода **MarianMT** от **Helsinki-NLP**.  
Приложение принимает текст от пользователя, выбирает нужную языковую пару и возвращает результат перевода через веб-интерфейс или API.

## Возможности
- перевод текста с **русского на английский**;
- перевод текста с **английского на русский**;
- веб-интерфейс для ручного ввода текста;
- API-метод для отправки текста и получения перевода в формате JSON;
- использование готовых моделей машинного перевода без ручного обучения.

## Как работает
При запуске приложения загружаются две предобученные модели и два токенизатора:
- `Helsinki-NLP/opus-mt-ru-en`
- `Helsinki-NLP/opus-mt-en-ru`

После этого приложение:
1. принимает текст и направление перевода;
2. токенизирует входную строку;
3. передаёт данные в модель MarianMT;
4. получает сгенерированный перевод;
5. возвращает результат пользователю.

## Используемые технологии
- **Python**
- **Quart**
- **Transformers**
- **PyTorch**
- **MarianMTModel**
- **MarianTokenizer**
- **Jinja2 / HTML templates**

## Маршруты приложения
### `GET /`
Главная страница с формой для перевода текста.

### `POST /`
Обработка текста из веб-формы и вывод перевода на странице.

### `POST /translate`
API-метод для перевода текста.  

## Поддерживаемые направления перевода
ru-en — русский → английский
en-ru — английский → русский

## Назначение проекта
Проект создан в учебных целях для практики:
работы с нейросетевыми моделями перевода;
интеграции моделей Hugging Face в Python-приложение;
создания веб-сервиса на Quart;
организации API для обработки текста.
